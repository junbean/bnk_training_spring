시험 웹페이지 기능 목록

1. 문제 조회 및 응시 화면 구성
	- 문제 5개 가져오기	=>	DB에서 순차적으로 5개 문제 조회
	- 시험페이지 출력	=>	JSP에서 문제 + 보기 4개 렌더링
	- 사용자 응답 수집	=>	선택한 보기 정보 수집(radio, form)
	- 타이머 구현		=>	제한 시간(예: 5분) 카운트다운, 시간 초과 시 자동 제출
2. 응답 제출 및 채점 처리
	- 답안 제출 처리	=>	사용자 선택값을 서버로 전송 (POST)
	- 정답 비교 및 채점	=>	DB에서 정답 조회 후 채점
	- 점수 계산		=>	총 5문제 중 정답 개수 세기
	- 결과 페이지 이동	=>	점수와 피드백 출력
3. 결과 페이지
	- 점수 출력		=>	"총 5문제 중 4문제 정답입니다" 형태 출력
	- 틀린 문제 확인	=>	사용자 응답 vs 정답 비교 내용 출력(테이블 형태)
	- 다시풀기/홈이동	=>	링크 버튼 제공

서버측 구성 요소
	- Controller
		/exam/start, 
		/exam/submit, 
		/exam/result 
	- DAO
		ExamQuestionDAO(문제 DB 접근)
	- DTO
		ExamQuestion(문제 정보), UserAnswerDTO(사용자 답변 전달용)
	- Mapper 
		exam-question.xml (SQL 정의 - MyBatis)
		
전체 흐름
	[사용자]
		→ /exam/start
	   		→ 문제 5개 출력 + 타이머 시작
	      	↓
			[제출 버튼] 또는 [시간 초과 자동 제출]
	      	↓
	 	→ /exam/submit (답안 POST)
	   		→ 채점 → 점수 계산
	      	↓
	 	→ /exam/result
	   		→ 점수 + 피드백 출력
